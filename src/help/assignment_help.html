Getting Started
To get to the assignment page you can click on the assignment icon on the home page. On the assignment page you can add a new assignment, view assignment count for each year, open an assignment or delete an assignment. In order to add an assignment, the course must be added first.
Add Assignment

(1) In the course field start typing the name or code of the course e.g. Calculus or MATH-100. you can also find a course by the term or year it was taking e.g. Winter or 2020. You should immediately see a drop-down list of course(s) matching your input.

(2) From the drop down select the course you want to add an assignment to. E.g. you can select "MATH-100 ==> Into to Calculus (Winter 2020)", or any other option from the drop-down list.

(3) Once the course name has being selected you can now enter the assignment name e.g. Group Project Report or My Awesome Presentation. Assignments name can be 25 max character in length.

(4) After adding the assignment name, click on file to select the assignment file you want to back up. On windows the file explorer will open.

(5) In the file explorer (for windows) browse to the file location and select your assignment file. After selecting your file click on ok. The file name should now be displayed as your selected file.

(6) Once your file has been selected, go ahead, and click on back up. For local back-up, your file would be saved in the location specified in the settings page.

(7) After clicking the back-up button you should now see a popup Success! dialogue box. Click ok and your assignment is now back-up and updated under the list of existing assignment. You can confirm the assignment was added successfully by viewing the list of assignment for that year.

View Assignment

(1) To view an added assignment or previously added assignment start by clicking on the year the assignment was taken in e.g. 2020. You will then see a drop list of assignment. (Years are displayed in descending order.)

(2) In the drop-down list you can view a list of all assignment added for that yea. From this drop down you will see information like semester, course code, name and if the assignment was backed-up to the cloud. You can open or delete an assignment file.

Open Assignment

(1) To open an assignment, start by clicking on the year the assignment was taken in e.g. 2020. You will then see a drop list of assignment. (Years are displayed in descending order.)

(2) In the drop-down list of assignment click on the open button.

(3) After clicking on open the assignment file should then open in its default file viewer. In our example "My Awesome Presentation" will open in Microsoft Words.

Delete Assignment

(1) To delete an assignment, start by clicking on the year the assignment was taken in e.g. 2020. You will then see a drop list of assignment. (Years are displayed in descending order.)

(2) In the drop-down list of assignment click on the delete button next to the assignment you want to remove. E.g. My Awesome Presentation.

(3) After clicking on the delete button you will then see a pop-up dialogue box. Click Yes to give permission to delete the assignment or no to cancel and return to the assignment page.

(5) You can confirm an assignment was successfully removed by viewing the list of remaining assignment for the year if it is the only assignment the year will be removed from the existing assignment list.
