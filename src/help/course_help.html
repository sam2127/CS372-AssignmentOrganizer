Getting Started
To get to the course page you can click on the course icon on the home page. On the course page you can add a course, edit a course, or delete a course.
ADD Course

(1) In the semester field start typing the semester term or year e.g. Winter or 2020. You should immediately see a drop-down list of all semesters matching your input.

(2) From the drop down select the semester you want to add a course to. E.g. you can select Winter 2020, or any other option from the drop-down list.

(3) Once the semester has being selected you can now enter the course code e.g. MATH-110. Course code must start with 2 alphabets and can only contain digits, letters, spaces, underscores, and dashes.

(4) Once the semester and course code are filled out you can now enter the course name e.g. Intro to Calculus. Course name must start with a letter alphabet and can only contain digits, letters, spaces, underscores, and dashes.

(5) Once the semester, course code and course name are filled you can now click on the Add button. You should now see a Success! dialogue box.

(6) In the Success dialogue box, you can now click ok and you course will be updated below the semester year. You can confirm by viewing the list of course for the semester year.

View Course

(1) To view an added course or previously added course click on the year the course was taken in. You will then see a drop list of courses. (Years are displayed in descending order.)

(2) In the drop list you can see all the courses added for the year grouped by the semesters for that year. From here you can see the assignment count for each course, edit a course and delete a course.

Edit Course

(1) To edit a course, start of by clicking on the year the course was taken. You will see a drop-down with the list of course.

(2) In the drop list you can see all the courses added for the year grouped by the semesters for that year. From here you can see the assignment count for each course, edit a course field and delete a course.

(3) In the drop list click on the edit button next to the course you want to edit e.g. Winter 2020: MATH-100: Intro to Calculus. After clicking the edit button, the box above will change from Add Course to Edit course.

(4) In the edit box you can choose to modify the semester, course code, course name or any combination of the three fields. In our case we will modify all three high lighted fields.

(5) Once you are finish editing the course you can go ahead and click up to proceed with the changes or cancel to remove all changes made.

(6) After clicking yes, you should then see a Success! dialogue box. From this box you will know your course has been successfully added. You can confirm by viewing the list of course for that semester year.

(7) After clicking yes, you should then see a Success! dialogue box. From this box you will know your course has been successfully added. You can confirm by viewing the list of course for that semester year.

Delete Course

(1) To delete a course, start of by clicking on the year the course was taken. You will see a drop-down with the list of course.

(2) In the drop list click on the delete button next to the course you want to delete e.g. Spring 2020: MATH-110: Calculus 1. You then see a pop-up dialogue box.

(3) In the pop up dialogue box click yes to continue deleting the semester or no to cancel deleting the semester and return to the course page.

(4) After clicking yes to delete the course, "course deleted successfully" will be displayed in the top right for a few seconds.

(5) You can confirm a course was successfully removed by viewing the list of remaining course for the year if it is the only course the year will be removed from the existing course list.
