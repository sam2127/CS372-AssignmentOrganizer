Getting Started
To get to the settings page you can click on the settings icon on the home page. On the setting page you can set the directory path on your local storage where you want to save your assignments. Changing the directory path will move all saved assignments to the new location as well.
Change Local Back-Up Directory

(1) By default, the local back-up directory is set to the root folder where the Assignment Organiser application is installed. In the example below the root directory is "D:\University Files\CS372-AssignmentOrganizer\assignments".

(2) To change the current storage location for your files, start by clicking on the change button.

(3) You should then see a pop dialogue box like the one below. click ok to proceed (Changing the location will move all assignments to your new directory) or no to cancel and return to the settings page.

(4) On windows the file explorer will pop-up. Browse to the new location you want to back-up your assignments to. In this example we will change from the default location to another folder in the same drive. After changing the path click on Select Folder.

(5) After clicking Select Folder you will then see a Success! dialogue box once the process has been successfully completed; click on ok. Note how ever the time this takes to display will vary depending on the number of saved assignments and individual assignment file size.

(6) After clicking ok you will also see a success notification in the top right-hand corner of the application.

(7) You should now see your Local Back-up Directory location set to the location it was changed to.

Reset Local Back-Up Directory

(1) The Reset button will start the process to change the local back-up directory path back to the assignments folder where the Assignment Organizer application installation files are.

(2) Click on the Reset button to start the reset process.

(3) After clicking on the reset button, you should now see a pop-up dialogue box like the one below. Click on Yes to continue moving the assignment back up to the default or No to cancel and return to the settings page.

(4) After clicking yes, you will then see a Success! dialogue box once the process has been successfully completed; click on ok. Note how ever the time this takes to display will vary depending on the number of saved assignments and individual assignment file size.

(5) After clicking ok you will also see a success notification in the top right-hand corner of the application.

(6) You should now see your Local Back-up Directory location set to its default location (where the Assignment Organiser installation files are saved).
